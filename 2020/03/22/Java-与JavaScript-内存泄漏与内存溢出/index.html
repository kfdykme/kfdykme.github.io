java内存回收机制

java中的内存主要指jvm虚拟机中的内存。通常一个java对象是被new或者通过放射的方式创建的。如果创建的是基础类型对象，则先在常量池中寻找，如果常量池中没有相对应得变量，则创建，然后栈中创建引用指向常量池

  • 监听器没有remove
  • 各种连接
  • 单例模式
